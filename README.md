# BeltEye Web Sitesi

Konveyör bant ek sağlığını yapay zeka ile izleyen BeltEye sisteminin tanıtım web sitesi.

## Yayınlama

Statik HTML — Vercel veya GitHub Pages ile doğrudan yayınlanabilir.

### Vercel

1. Vercel'de "New Project" → GitHub repo'yu seçin
2. Framework Preset: **Other**
3. Build Command: boş
4. Output Directory: `.`
5. Deploy

### GitHub Pages

Settings → Pages → Source: `main` branch, `/` (root) → Save
